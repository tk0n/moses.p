# moses ~ Automated Snapshots for [PersonaIam/persona-mainnet](https://github.com/PersonaIam/persona-mainnet)

## Installation

```bash
git clone https://github.com/tk0n/moses.p
cd ~/moses
bash moses.sh install
bash moses.sh start
```
